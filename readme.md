# Struts2Scanner

Struts2Scanner is a vulnerability scanner to find out if a target endpoint is vulnerable to Remote Code Execution. Currently it checks against following vulnerabilities.

* cve-2020-17530
* cve-2019-0230
* cve-2018-11776 
* cve-2017-5638
* cve-2017-9791
